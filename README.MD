# A Quick to Review Flask Framework 
This is a Python project to create my blog website using Flask in Python. Still on progress. 
## How to run 
1. Clone the repository
2. Create and activate the virtual environment
3. Install all dependencies, see requirements.txt
4. Run the run.py script
## Project Structure
